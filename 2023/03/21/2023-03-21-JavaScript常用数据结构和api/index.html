字符串

常见的 JavaScript 字符串方法：

  1. length：获取字符串的长度。

  2. indexOf 和 lastIndexOf：查找指定子字符串在字符串中的位置。

  3. charAt 和 charCodeAt：获取指定位置的字符和字符的 Unicode 编码。

  4. substring 和 substr：提取字符串中的子字符串。

  5. slice：提取字符串的部分内容，并返回一个新字符串。

  6. toUpperCase 和 toLowerCase：将字符串转换为大写或小写。

  7. trim：去除字符串两端的空格。

  8. split：将字符串分割成子字符串数组。
